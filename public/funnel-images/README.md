# Funnel Images

This folder contains placeholder images for the funnel copy page.

## Recommended Images

Add the following images to this folder for optimal funnel display:

1. **placeholder-video.jpg** - Video thumbnail image (1920x1080px recommended)
2. **hero-bg.jpg** - Hero section background image
3. **product-1.jpg** - Product/service showcase image
4. **product-2.jpg** - Secondary product image
5. **testimonial-1.jpg** - Client testimonial photo
6. **testimonial-2.jpg** - Client testimonial photo
7. **testimonial-3.jpg** - Client testimonial photo

## Image Guidelines

- Use high-quality images (at least 1000px wide)
- Keep file sizes optimized for web (under 500KB per image)
- Use JPG for photos, PNG for graphics with transparency
- Ensure images are relevant to your business/offer

## Example Funnel Image Sources

- Unsplash (https://unsplash.com) - Free high-quality images
- Pexels (https://pexels.com) - Free stock photos
- Your own product photos
- Client photos (with permission)
